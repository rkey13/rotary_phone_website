How to Use the Archive
  • Claim a Number: Enter any 1 to 7 digit number. If it is empty, you claim it by making the first audio submission!
  • The Password System: If you provide a password when claiming a number, you become its owner. If you leave it blank, it becomes an open "voicemail" where anyone can leave a recording.
  • Locking a Line: Checking the "Lock number" box freezes the line. Once locked, only the owner (using the correct password) can submit new audio.
  • Active Deployments: If a number has multiple recordings, the "Active Deployment" is the specific file that will play when someone dials this number on the physical rotary phone.
